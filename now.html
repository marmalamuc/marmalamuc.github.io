Other goings on

I bought a new computer, and have been spending most of my time migrating to it. I tried to automate a lot of the process with Homebrew & Mackup (a command line utility that backs up dotfiles), but Z shell (an alternative to Bash for Linux OSs) wouldn’t run my install script, and Mackup fucked up the path variables in my profile—so I ended up spending more time cleaning up those mistakes than if I had just used the migration assistant.
