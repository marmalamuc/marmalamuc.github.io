Athletics

I’ve been following the program the fitness manager at my BJJ gym writes for his group powerlifting class. It’s sort of a wave program, which I’ve never seen before. Week-to-week, the set intensity undulates; the 12-week rep scheme look something like: 12-10-8-6-12-8-6-4-10-6-4-MAX

Wave sets are my favorite way to break through plateaus, but those are performed in an individual session. Alternating high intensity sets with relatively low intensity sets produces an effect called “post-tenatic potentiation”; essentially, high force production recruits more muscle fibers for even higher force production (if you get the timing & load schemes right). I’m interested to see if/how that plays out over a period of weeks.

Learning Spanish

Current Duolingo streak: 1,325 days (as of 07 Feb 2022 )

Been reviewing the past perfect lately (have + past participle). I’ve used it before in the group classes I used to take, but I really like Duolingo’s method of introducing it without asking you to think about the prescriptive theory of it. It’s less academic, which is great because I just want to speak, be understood, & understand; like English, I don’t care what a style guide or the Royal Spanish Academy prescribes.

Technology

Symlinks have saved my sanity. After days of dicking around with Git submodules (and even considering MAMP), I finally looked at symlinks. It’s an elegant solution to my single-sourcing issue; and, as it turns out, the simplest of all the options I had been considering. A single command replaced what would have been seven Git commands each time I updated my submodules for each project that contained that submodule. I hate Git so much.

I started learning JavaScript. I definitely don’t have the right kind of brain for this, but if I can memorize solutions for given situations, I can really enhance this website & my booksite.
